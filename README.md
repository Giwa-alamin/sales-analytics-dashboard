# 📊 Sales Analytics Dashboard

A professional interactive sales analytics dashboard built with Python, Streamlit, Pandas and Plotly.

## Features

- Revenue KPI
- Orders KPI
- Customer KPI
- Revenue by Product
- Revenue by Category
- Revenue Trend
- Interactive Filters
- Responsive Dashboard

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- GitHub

## Dataset

Sample retail sales dataset.
